# Java OOP Concepts

This project demonstrates core Object-Oriented Programming concepts in Java.

## Oops Concepts
- local and local varibles
- Abstraction
- Inheritance
- Polymorphism
- Interfaces
- Inner Classes
## Exception Handling
- Try-Catch basics
- Handling ArithmeticException
- Default Exception handling by JVM
## Arrays
- 1D Arrays
- 2D Arrays
- Jagged Arrays
- Enhanced for loop
- ArrayIndexOutOfBoundsException
